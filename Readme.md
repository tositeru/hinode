# Hinode Unity

このライブラリはUnity用の開発支援ライブラリとなります。

Unity 2019.2以降のバージョンに対応しています。

## ライセンス

Apache Licence Version 2.0

## インストール

UnityのPackage Managerを使用しGit URLからインストールできます。
